# ESG Score vs Financial Performance Analysis

## Overview

This project investigates the relationship between Environmental, Social, and Governance (ESG) scores and financial performance metrics of S&P 500 companies.

The objective is to determine whether companies with higher ESG ratings generate superior returns or lower investment risk.

## Business Problem

As ESG investing becomes increasingly popular, investors need evidence on whether ESG performance contributes to financial performance and risk reduction.

## Dataset

Source:
S&P 500 ESG and Stocks Data (2023-24)

Data Includes:

- ESG Scores
- Environmental Score
- Social Score
- Governance Score
- Controversy Scores
- Daily Stock Prices
- Industry Information

## Methodology

1. Data Cleaning
2. Exploratory Data Analysis
3. Correlation Analysis
4. OLS Regression
5. K-Means Clustering
6. Principal Component Analysis (PCA)
7. Sector Analysis

## Key Findings

- ESG scores showed weak correlation with stock returns.
- Higher ESG companies demonstrated slightly lower volatility.
- Financials, Technology, and Healthcare sectors contained more high-ESG firms.
- ESG metrics appear more useful for risk screening than return prediction.
